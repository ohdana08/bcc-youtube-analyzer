BCC INSIGHT — 운영 진단
    본 도구 안내. 비즈니스커리어컨설팅(BCC)이 YouTube Data API v3를 기반으로 독자 개발한 채널 분석 도구입니다. 분석 알고리즘, 인사이트 멘트, UI/UX 디자인은 모두 BCC의 고유 자산입니다. 입력된 API Key는 사용자의 브라우저에만 저장되어 외부로 전송되지 않습니다.
    문의: 비즈니스커리어컨설팅 (BCC)

    YouTube API Key 발급 방법

    1. Google Cloud Console에 접속합니다
      (console.cloud.google.com)
    2. Google 계정으로 로그인합니다
    3. "Google Cloud 시작하기" 클릭 → 약관 동의 → 결제 카드 등록(무료 한도 내 청구 없음)
    4. 좌측 상단 "프로젝트 선택" → "새 프로젝트" 생성
    5. 좌측 메뉴 → "API 및 서비스" → "라이브러리"
    6. 검색창에 "YouTube Data API v3" 입력 → "사용 설정"
    7. "사용자 인증 정보" → "+ 사용자 인증 정보 만들기" → "API 키"
    8. 생성된 키를 복사 → 좌측 사이드바에 붙여넣고 "저장" 클릭
    💡 무료 할당량: 하루 10,000 unit (채널 분석 1회당 ~3 unit)
    💡 Key는 브라우저에만 저장되며 외부로 전송되지 않습니다
    ⚠️ finder(잘 나가는 영상 찾기)에서 이미 저장하셨다면 같은 키가 자동으로 불러와집니다.
