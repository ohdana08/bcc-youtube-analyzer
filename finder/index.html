YouTube API Key 발급 방법

  1. Google Cloud Console에 접속합니다
    (console.cloud.google.com)
  2. Google 계정으로 로그인합니다
  3. "Google Cloud 시작하기" 클릭 → 약관 동의
  4. ⚠️ 결제 카드 등록필수 단계
    • Google 보안 정책상 카드 등록이 필요합니다
    • 무료 한도 내에서는 절대 요금이 청구되지 않습니다
    • 신용/체크카드 모두 가능
    • 본인 인증 SMS 받기
  5. 좌측 상단 "프로젝트 선택" → "새 프로젝트" 생성
    (이름 입력 후 만들기)
  6. 좌측 메뉴 → "API 및 서비스" → "라이브러리"
  7. 검색창에 "YouTube Data API v3" 입력 → "사용 설정"
  8. "사용자 인증 정보" → "+ 사용자 인증 정보 만들기" → "API 키"
  9. 생성된 키를 복사 → 좌측 사이드바에 붙여넣고 "저장" 클릭
💡 무료 할당량: 하루 10,000 unit (검색 약 100회 가능)
💡 Key는 브라우저에만 저장되며 외부로 전송되지 않습니다
⚠️ 가장 어려운 단계는 4번(카드 등록)입니다. 천천히 진행하시면 됩니다. 요금은 절대 청구되지 않으니 안심하세요.
💡 실제 안전장치 설정강력 추천

요금이 청구될까 봐 불안하시면 다음 안전장치를 설정해두세요:

[Google Cloud Console에서]
  1. 좌측 메뉴 → "결제"
  2. "예산 및 알림" 클릭
  3. "예산 만들기"
  4. 예산 금액: 1,000원 또는 0원 설정
  5. → 1원이라도 청구되면 즉시 알림 수신
  6. → 미리 확인 가능, 100% 안심
